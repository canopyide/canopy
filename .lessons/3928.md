## Context

Three independent issues were bundled here: dnd-kit bounding box staleness during drags, macOS back-navigation gesture triggering on dock scroll, and xterm terminals waking from hibernation with stale dimensions.

## Problem

### 1. Stale DnD measurements

`DndContext` uses dnd-kit's default measurement strategy, which caches droppable bounding boxes. The grid applies a 200ms CSS transition on `grid-template-columns`. During a window resize mid-drag, column layout changes but cached boxes are stale — panels drop into wrong slots.

### 2. macOS navigation gesture

The dock scroll container uses `overflow-x: auto` but lacks `overscroll-behavior-x: none`. On macOS, horizontal trackpad scrolling past the scroll boundary triggers Electron's back/forward navigation gesture.

### 3. Post-wake stale dimensions

`handlePostWake` used `latestCols`/`latestRows` — values cached before hibernation. After wake, if the window was resized while hibernated, terminals would have incorrect dimensions until the next resize event. The post-wake path also only triggered for alt-screen terminals, not all terminals.

## Root Cause

dnd-kit defaults to measuring only when dragging starts. The `TerminalRendererPolicy` alt-buffer gate on `onPostWake` was overly narrow.

## Lesson

- Configure `MeasuringStrategy.Always` with `frequency: 150` in dnd-kit's `DndContext` whenever any animated layout change (CSS transitions on grid, panel resize) can occur during a drag.
- Always set `overscroll-behavior-x: none` on any horizontally scrollable container in an Electron app on macOS.
- After hibernation wake, call `fit()` before falling back to cached `latestCols`/`latestRows`. Skip `fit()` only for settled-strategy agents that need atomic xterm+PTY resize.
- The post-wake callback should fire for all terminals, not just alt-screen ones.

## Applies when

- Adding new drag-and-drop droppable regions in animated layouts
- Adding horizontal scroll containers in Electron on macOS
- Implementing terminal wake/restore logic after any off-screen or hibernated period
