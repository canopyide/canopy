## Context

Agent terminals are launched with per-agent CLI flags (e.g., `--dangerously-skip-permissions` for Claude, `--yolo` for Gemini). These flags are stored in renderer-side Zustand state and persisted to `appState.terminals` snapshots. The PTY host (`TerminalPublicState`) had no knowledge of these flags.

## Problem

After an app crash, two paths lead to flag loss:

1. **Orphaned PTY recovery ("Start fresh"):** `resetToFresh()` wipes `appState.terminals`. Orphaned PTY processes are rediscovered by hydration, but `buildArgsForOrphanedTerminal()` reconstructs terminal instances from `TerminalPublicState` — which contained no `agentLaunchFlags`. Restarting the agent uses no flags.

2. **Restore path with debounce race:** If the crash occurred within the persistence debounce window, `agentLaunchFlags` were never flushed to disk. The backup snapshot has no flags; the restored terminal has none.

## Root cause

`agentLaunchFlags` and `agentModelId` were renderer-only state. The PTY host, which survives crashes, had no record of them. Crash recovery is the only path where renderer state is unavailable and PTY state is the sole source of truth.

## Lesson

Any per-terminal configuration that must survive crashes must be stored in `TerminalPublicState` (the backend PTY layer), not only in renderer persistence. Renderer persistence is subject to debounce delays and full wipe on "Start fresh" recovery. The fix involves threading flags through the full PTY state pipeline (TerminalProcess → PtyClient → pty-host IPC → snapshot endpoints) and updating `buildArgsForOrphanedTerminal()` to pull flags from backend state with a fallback to current agent settings for pre-fix terminals.

## Applies when

- Adding new per-terminal configuration that must survive crashes
- Implementing any agent session property that needs to be restored on reconnect
- Debugging "agent missing flags after crash" or "unexpected permission prompts after restart"
- Designing state that lives at the boundary between renderer persistence and PTY host state
